# scripts

バックエンド用のスクリプト集

## generate-dummy-players.ts

100件のダミープレイヤーデータを生成します。

### 使い方

```bash
cd packages/backend
pnpm seed:players
```

### 生成されるデータ

- **プレイヤー数**: 100件
- **名前**: 日本人の一般的な苗字・名前の組み合わせ
- **ニックネーム**: 一部のプレイヤーにランダムに設定（50%の確率）
- **学年タグ**: 小学1年生〜6年生からランダムに割り当て
- **親ユーザー**: チーム内の最初のユーザーを使用

### 前提条件

- Supabaseがローカルで起動していること（`pnpm db:start`）
- チームとユーザーが既に作成されていること
- 学年ラベルと学年タグが作成されていること（マイグレーション済み）

### サンプルデータ

```
✅ Successfully created 100 players!

📊 Sample created players:
  - 木村 拓海 (2年生)
  - 田中 樹 (6年生)
  - 井上 大輝 (1年生)
  - 佐々木 蒼 (2年生)
  - 加藤 優奈 (4年生)
```
