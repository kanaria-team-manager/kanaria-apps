# @kanaria/shared

This package contains shared TypeScript types, interfaces, and utility functions used by both the frontend and backend applications.

## Usage

This package is installed as a workspace dependency in both `@kanaria/frontend` and `@kanaria/backend`.

## Scripts

- **`pnpm build`**: Compiles the TypeScript code.
- **`pnpm dev`**: Runs the TypeScript compiler in watch mode.

# @kanaria/shared

フロントエンドとバックエンドのアプリケーション双方で使用される、共有のTypeScriptの型、インターフェース、ユーティリティ関数を含むパッケージです。

## 使用方法 (Usage)

このパッケージは、`@kanaria/frontend` と `@kanaria/backend` の両方でワークスペースの依存関係としてインストールされます。

## スクリプト (Scripts)

- **`pnpm build`**: TypeScriptコードをコンパイルします。
- **`pnpm dev`**: TypeScriptコンパイラをウォッチモードで実行します。
